## 機能
7-Zip File Manager、GUI、SFXのダイアログに設定されているフォントを書き換える。

"8pt MS Shell Dlg" から "10pt Yu Gothic UI"に変更する。

## 使い方
7-Zipをインストールしたフォルダに7zFontChange.exeをコピーして、管理者として実行する。
